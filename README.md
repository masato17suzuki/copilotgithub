## 試行錯誤

コーディングが800行以上になり正確なファイル出力が不可能と判断。セッションをリセットして以下のプロンプトを作成した。
あなたはPythonエンジニアです。Windows上で動作するシンプルなGUI付き自動化ツールを作成してください。
目的:

ハローワーク求人票PDFから宛先メールアドレスを抽出
既定の履歴書・職務経歴書を添付
紹介状をユーザーが選択
メール本文と件名を確認・編集
送信前に送信先・添付ファイルを確認
SMTPでメール送信

要件:

ファイルは main.py の1ファイルだけ
GUIは tkinter を使用
PDF解析は PyPDF2 を使用
設定ファイル config.json でパスとSMTP情報を保存
実行は pythonw でCMDウィンドウなし


これで①メール送信成功（GmailはアプリパスワードI6桁を設定）②履歴書・職務経歴書の保存が実現。

## 完成版の使い方

## 事前準備（初回のみ）

Pythonがインストールされていない場合は [python.org](https://www.python.org/downloads/) からダウンロードしてインストールしてください。
**インストール時に「Add Python to PATH」にチェックを入れることが必須です。**

次にキーボードで `Windowsキー + R` を押して `cmd` と入力しEnter。黒い画面が開いたら以下を貼り付けてEnterを押します。
pip install PyPDF2

`Successfully installed` と表示されれば準備完了です。

### 1. 起動ファイルの作成

メモ帳を開き以下を貼り付け、**ファイルの種類を「すべてのファイル」**にして `main.bat` という名前で保存（.txtにすると失敗します）。

```bat
@echo off
start pythonw main.py
```
### 3. 動作確認

UIが起動したら、履歴書・職務経歴書・SMTP情報を一度設定するだけで次回から自動読み込みされます。Gmailはアプリパスワード16桁が必要です。
